# Component Specifications

## Hero Component

Props: HeroData
Calls: /api/hero

Layout:
- Left: Headline + Description + CTAs
- Right: Dashboard Stats

## Products Component

Props: ProductsData
Calls: /api/products

Layout:
- Featured (2 cols): GOFLOW AI, Political
- Coming Soon (3 cols): Rider, Market, Agri

## Enterprise Core Component

Props: EnterpriseCoreData
Calls: /api/enterprise-core

Show: Architecture diagram

## Why GOFLOW Component

Props: WhyGoflowData
Calls: /api/why-goflow

Show: 4 feature cards

## Business Model Component

Props: BusinessModelData
Calls: /api/business-model

Show: Revenue streams (40%, 25%, 20%, 10%, 5%)

## Investment Component

Props: InvestmentData
Calls: /api/investment

CTA: Request Deck, Schedule Meeting
